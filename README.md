### はじめに

ソース取得後、ルートディレクトリにて下記を実行

```
npm install
```

### 開発

[開発用ブックマークレット](docs/bookmarklet/local.js)を登録する

下記を実行

```shell
npm run dev
```

### ビルド

```shell
npm run build
```
